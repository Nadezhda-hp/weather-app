# Прогноз погоды

Веб-приложение для просмотра прогноза погоды с использованием HTML, CSS и JavaScript.

## Реализованные функции

- Запрос геолокации при первом открытии
- При отклонении геолокации — форма ввода города
- Прогноз погоды на сегодня + 4 дня вперёд
- Добавление дополнительных городов (без ограничений)
- Выпадающий список при вводе города (автокомплит через Open-Meteo Geocoding API)
- Валидация города: нельзя добавить несуществующий
- Кнопка «Обновить» — новые HTTP-запросы без перезагрузки
- Состояния интерфейса: загрузка (спиннер), ошибка, успех
- Сохранение городов в localStorage, восстановление при перезагрузке
- Удаление городов
- Адаптивный дизайн (десктоп + мобильные)

## Используемые API

- [Open-Meteo Weather API](https://open-meteo.com/) — бесплатный, без ключа
- [Open-Meteo Geocoding API](https://open-meteo.com/en/docs/geocoding-api) — поиск городов

## Технологии

- HTML5
- CSS3 (без фреймворков)
- Vanilla JavaScript (без фреймворков)
- Fetch API для HTTP-запросов
